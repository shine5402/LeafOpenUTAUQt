◆ getImage()

QImage VoiceBank::getImage ( ) const

获得该声库的图标。

音源库的图标是指在 character.txt 的 image 字段中定义的音源的图标。在 character.txt 中的值为其路径。 VoiceBank 会将其读取到 QImage 实例中。
建议音源库的图标尺寸为 100*100 ，或至少宽高比为 1:1 。

返回
包含音源图标的 QImage 。
参见
setImage(const QImage& _image, const QString &newImageFileName)
