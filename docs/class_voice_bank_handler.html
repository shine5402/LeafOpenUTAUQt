详细描述

用于管理一系列VoiceBank。

VoiceBankHandler 是用于管理 VoiceBank 的类。
VoiceBankHandler 采用单件模式，您不能直接创建，而是通过 getVoiceBankHandler() 来获取其实例。
一般来说，您不需要手动添加 VoiceBank ，而是使用 readVoiceBanksFromMonitorFolders() 来从用户指定的监视文件夹中读取音源库，并自动转化为 VoiceBank 的实例。
它允许在它管理的 VoiceBank 中通过名称、分类、标签等条件查找音源库，详见 findIDByNameOrPath(const QString &text) const 、 findIDByCategory(const QString &category) const 、 findIDByLabel(const QString &label) const 。
它也允许对内部的 VoiceBank 列表进行排序，详见 sort(SortableInformationID sortWhat, Qt::SortOrder order) 。

◆ isUseOldFolderScan()

bool VoiceBankHandler::isUseOldFolderScan ( )

返回 VoiceBankHandler 是否使用旧式文件夹扫描策略

Leaf Open UTAU Qt 这部分代码的早期版本中，将监视文件夹的子文件夹直接作为 VoiceBank 的路径。这种策略与 UTAU 相同，但是在某些场景下不够实用。所以在较新版本的代码中， VoiceBankHandler 将会递归查找监视文件夹的子文件夹来确定音源库所在的文件夹。
VoiceBankHandler 使用 VoiceBank::isVoiceBankPath(const QString &path) 来确定一个文件夹是否是一个音源库文件夹。

返回
如果使用旧式扫描策略，该函数返回 true ，否则返回 false 。
